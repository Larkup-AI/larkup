title: "API Reference Overview"
description: "Interact with your generated RAG Server over HTTP."
---

Once you launch your generated server (e.g., via `larkup serve` or Docker), it exposes a set of RESTful endpoints.

These endpoints allow you to integrate Larkup into your existing products, web apps, or chatbots.

## Base URL
By default, the server runs on `http://localhost:8080`.

## Authentication
If you generated the server with basic defaults, the endpoints are open. However, if your endpoints rely on embedding models (like OpenAI), ensure you have set up the required API keys in the Configuration page.
